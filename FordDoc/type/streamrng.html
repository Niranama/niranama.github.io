StreamRNG

type, public, ABSTRACT, extends(BaseRNG) :: StreamRNG

The StreamRNG type is an abstract PRNG type that directly extends the BaseRNG type. It provides an additional API for a PRNG that can be used as a stream PRNG. Therefore, all so-called Stream PRNGs should extend from this base type.
