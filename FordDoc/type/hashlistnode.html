HashListNode

type, public, ABSTRACT, extends(DoublyLinkedNode) :: HashListNode

The HashListNode type is a doubly-linked-list node type intended to be used in conjunction with the IntrusiveHashList type. Similar to other intrusive containers, the IntrusiveHashList type requires a user to define a new (concrete) node type that extends the HashListNode type where the new node type contains a key and its associated value as its components.
The following code snippet illustrates how to define a new node type. 


  TYPE, EXTENDS(HashListNode) :: AssociativeNode
      CHARACTER(LEN=:), ALLOCATABLE   :: Key      ! stored string key
      INTEGER                         :: Value1   ! stored integer value
      REAL                            :: Value2   ! stored real value
  END TYPE

The HashListNode type is a subtype of the DoublyLinkedNode type and it defines several additional methods required by the IntrusiveHashList type. Therefore, a user must implement these deferred procedures in addition to other deferred procedures required by all its parent types.
