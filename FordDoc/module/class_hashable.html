Hashable

PURPOSE OF THIS MODULE:
This module contains the Hashable type and its related routines. The Hashable type is an abstract data type representing an object that can be converted to a hash value. The Hashable type and its concrete extended types are intended to be used with hash tables.
A concrete extended type that extends the Hashable type must implement the ComputeHashValue deferred procedure. Because the the Hashable type is a subtype of the Comparable type, all extended types must also implement those deferred procedures required by a subtype of the Comparable type.
